# Proyecto MEGATRONIX

**Autores del proyecto**: Alejandro Maman && Miguel Cabrera

---
### Compilar
```bash
gcc -o out Memsym Memsym.c
```

---
### Ejecutar
```bash
./Memsym > logcache.txt # para guardar el out en logcache.txt
./Memsym # para ver la ejecución a tiempo real (con sus sleeps)
```
